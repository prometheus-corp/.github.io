ABOUT PROMETHEUS

Prometheus is an evolutionary keyboard that will optimize consumer interaction with their computers. The keyboard is multifaceted with the ability to quickly turn into a touchscreen display allowing the user to interact with a personally customizable interface with unlimited design layout possibilities and button utilities.

What is it?

TWO MODES

MECHANICAL MODE

In the keyboards mechanical mode users can

  • Use the knob
  • Use the slider
  • Re-Map Every Button/Knob/Slider to any action
  • Add Custom Backlighting
  • Fully Functional Keyboard

TOUCHSCREEN MODE

In the keyboards touchscreen mode users can

  • Use the touchscreen
  • Interact with the touchscreen anyway
  • Re-Map Every virtual Button/Knob/Slider on the screen to any action
  • Use it as a screen extension
