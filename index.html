ABOUT PROMETHEUS

Prometheus is an evolutionary keyboard that will optimize consumer interaction with their computers. The keyboard is multifaceted with the ability to quickly turn into a touchscreen display allowing the user to interact with a personally customizable interface with unlimited design layout possibilities and button utilities.
